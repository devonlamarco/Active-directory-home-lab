# Lab 6 – Delegation of Control

## Objective

Delegated administrative permissions to a Help Desk security group using the Delegation of Control Wizard to implement least-privilege administration.

---

## Environment

- Windows Server
- Active Directory Users and Computers (ADUC)
- Active Directory Domain Services (AD DS)

---

## Tasks Completed

- Created a HelpDesk security group
- Added an IT user to the HelpDesk group
- Used the Delegation of Control Wizard
- Delegated password reset permissions
- Reviewed delegated permissions in the OU security settings
