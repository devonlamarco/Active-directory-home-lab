# Lab 7 – NTFS Permissions

## Objective

Configured NTFS permissions on departmental folders using Active Directory security groups to implement role-based access control.

---

## Environment

- Windows Server
- Active Directory
- NTFS File System

---

## Tasks Completed

- Created departmental folders
- Reviewed default NTFS permissions
- Disabled inheritance
- Assigned security groups to folders
- Configured Modify permissions
- Reviewed Advanced Security Settings

---

## Folder Structure

```text
C:\CompanyData
├── IT
├── HR
├── Accounting
└── Sales
```
