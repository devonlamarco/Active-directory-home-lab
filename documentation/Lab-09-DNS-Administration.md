# Lab 9 – DNS Administration

## Objective

Configured and verified DNS records in an Active Directory-integrated DNS zone by creating A and CNAME records and validating name resolution.

---

## Environment

- Windows Server
- Active Directory Domain Services
- DNS Server

---

## Tasks Completed

- Explored Forward Lookup Zones
- Reviewed Active Directory DNS records
- Created an A Record
- Created a CNAME Record
- Verified DNS resolution with nslookup

---

## Commands Used

```cmd
ipconfig

nslookup fileserver.devonlab.local

nslookup files.devonlab.local
```
